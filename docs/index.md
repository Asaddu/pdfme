## Getting Started

### Quick Start

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm run build

# Run tests
pnpm run test

# Start playground
cd playground && pnpm run dev
```

### Development Mode

```bash
# Run dev mode in packages you're editing
cd packages/common && pnpm run dev
cd packages/schemas && pnpm run dev
cd packages/ui && pnpm run dev

# Run playground to test changes
cd playground && pnpm run dev
```
