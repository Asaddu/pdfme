# PDFme Project Overview

## Executive Summary

PDFme is an open-source TypeScript-based PDF generation and manipulation library for web and Node.js applications. It provides a complete solution for creating, designing, and manipulating PDFs with a focus on performance, minimal dependencies, and ease of use.

**Repository Type:** Monorepo (pnpm workspaces)
**Primary Language:** TypeScript
**License:** MIT
**Homepage:** https://pdfme.com

## Project Purpose

PDFme enables developers to:
- Generate PDFs from JSON templates
- Design PDF templates using a visual WYSIWYG editor
- View and fill PDF forms interactively
- Manipulate existing PDFs (merge, split, rotate)
- Convert between PDF formats

## Key Features

| Feature | Description |
|---------|-------------|
| Fast PDF Generator | Works on Node.js and in the browser using templates |
| Visual Template Designer | WYSIWYG editor for creating PDF templates |
| JSON-Based Templates | Templates are simple JSON data structures |
| Plugin Architecture | Extensible schema system for custom field types |
| Cross-Platform | Works in both Node.js and browser environments |
| Font Support | Full font embedding with CJK character support |

## Quick Start

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm run build

# Run tests
pnpm run test

# Start playground for development
cd playground && pnpm run dev
```
